## 🚀 Công nghệ sử dụng
- **Frontend**: HTML5, CSS3, JavaScript (Vite)
- **Backend & Database**: Supabase

## 🛠 Cài đặt & Chạy dự án
1. Cài đặt các dependencies:
   ```bash
   npm install
   ```
2. Chạy môi trường development (với Vite):
   ```bash
   npx vite
   ```

## 📝 Script khác
- Chạy test:
   ```bash
   npm run test
   ```

## 🔐 Đăng nhập
Mỗi thành viên có một mã đăng nhập riêng ban đầu. Vui lòng xem thông tin chi tiết hoặc liên hệ để nhận mã, sau lần đầu có thể thay đổi mã tùy ý.
